# Project Name

Club Challenge

## Installation

Prerequisites:
Microsoft SQL Server 2008R2 or newer (for best results use 2012)
Visual Studio Professional 2010 or newer
ASP.NET Platform 5
ADO.NET and EF Tools for Visual Studio
